# Eenvoudige Klantenbeheer Applicatie

Ontwikkel een eenvoudige consoleapplicatie in VB.NET die basisfunctionaliteit biedt voor het beheren van klanten. De applicatie moet het mogelijk maken om nieuwe klanten toe te voegen, bestaande klanten te bekijken en klanten te zoeken op naam.  Focus op het correct toepassen van klassen, objecten en referentietypes.  Zorg voor een duidelijke structuur en gebruik van namespaces.

## Opdracht

Implementeer de gevraagde functionaliteit.
